# Aerie Roofing - Professional Roofing Services

A modern, responsive website for a professional roofing company.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional interface with gradient backgrounds
- **Service Showcase**: Highlight key roofing services
- **Portfolio Section**: Display completed projects
- **Contact Form**: Easy way for customers to get in touch
- **Navigation**: Smooth scrolling navigation bar

## Sections

1. **Hero Section**: Eye-catching introduction with call-to-action button
2. **Services**: Four main service offerings with cards
3. **Portfolio**: Gallery of recent projects
4. **Contact**: Contact form for inquiries

## Files

- `index.html` - Main HTML file with page structure
- `styles.css` - Styling and responsive design
- `README.md` - Documentation

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Customize the content with your company information

## Customization

### Update Company Name
Replace "Aerie Roofing" throughout the HTML file with your company name.

### Add Portfolio Images
Replace the "Project X" placeholders with actual project images and descriptions.

### Modify Colors
Edit the color values in `styles.css` to match your branding:
- Primary color: `#1e3c72`
- Secondary color: `#667eea`
- Accent color: `#ff6b6b`

### Add Functionality
To make the contact form functional, you'll need to add a backend service or integrate with a third-party form handler.

## Deployment

This website can be deployed using:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service
